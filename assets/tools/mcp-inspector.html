ⓘ Troubleshooting

HTTP 401 / 403Missing or invalid token. Paste a valid Bearer token; it is sent only for the current request.
Blocked targetLocalhost, private IPs and metadata addresses are rejected by the Worker by design. Test those locally.
Empty tools listSome servers require initialize first. Click Initialize, then List tools so the session id is reused.
Timeout / 504The remote server took too long. Increase the request timeout or check that the endpoint is reachable.
Local / stdio server? Generate the npx command

Remote HTTP servers are tested live in the tab above. A local stdio server (for example mcpgateway.wrapper) can only be inspected on your own machine — this just builds the command for you to paste into your terminal.
